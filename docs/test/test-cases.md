# Тест-кейсы: Project Planner

## **Создание задачи**

### ID: TC-PP-001
**Заголовок:** Создание новой задачи

**Предусловие:** Главная страница открыта, задач нет

| Шаг | Ожидаемый результат |
|-----|----------------------|
| Нажать кнопку "Создать" в хедере | Открывается модальное окно создания задачи |
| Ввести название задачи в поле "Название задачи" | Текст отображается в поле |
| Ввести описание в поле "Описание" | Текст отображается в поле |
| Выбрать приоритет "Высокий" из выпадающего списка | Выбран "Высокий" |
| Нажать кнопку "Создать" | Модальное окно закрывается, задача появляется в колонке "Очередь" |

---

## **Редактирование задачи**

### ID: TC-PP-002
**Заголовок:** Редактирование существующей задачи

**Предусловие:** Задача существует в колонке "Очередь"

| Шаг | Ожидаемый результат |
|-----|----------------------|
| Нажать на карточку задачи | Открывается модальное окно редактирования задачи |
| Изменить название задачи | Новое название отображается в поле |
| Изменить приоритет на "Средний" | Выбран "Средний" |
| Нажать кнопку "Сохранить" | Модальное окно закрывается, карточка задачи показывает обновленные данные |

---

## **Перетаскивание задач**

### ID: TC-PP-003
**Заголовок:** Перемещение задачи между колонками с помощью drag-and-drop

**Предусловие:** Задача существует в колонке "Очередь"

## **Удаление задачи**

### ID: TC-PP-004
**Заголовок:** Удаление задачи с помощью кнопки удаления

**Предусловие:** Задача существует в любой колонке

| Шаг | Ожидаемый результат |
|-----|----------------------|
| Нажать кнопку ✖ на карточке задачи | Задача исчезает из колонки |
| Счетчик в заголовке колонки уменьшается | Отображается новое количество |
| Обновить страницу | Задача не появляется снова |

## **Контекстное меню**

### ID: TC-PP-005
**Заголовок:** Использование контекстного меню для операций с задачами

**Предусловие:** Задача существует в колонке "Очередь"

| Шаг | Ожидаемый результат |
|-----|----------------------|
| Правый клик на карточке задачи | Открывается контекстное меню рядом с курсором |
| Выбрать "Вырезать" из меню | Задача исчезает, копируется в буфер обмена |
| Правый клик на пустом месте | Открывается контекстное меню |
| Выбрать "Вставить" из меню | Задача появляется в текущей колонке |

## **Экспорт/Импорт задач**

### ID: TC-PP-006
**Заголовок:** Экспорт задач в JSON файл

**Предусловие:** Существует хотя бы одна задача

|-----|----------------------|

| Подтвердить импорт | Задачи появляются на доске, страница перезагружается |
